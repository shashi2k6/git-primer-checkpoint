Practice with git and github.com

Cloned from gSchool/git-primer-checkpoint.


1. git init
2. git add .
3. git commit -m "message"

5. git push -u origin master

6. git clone <url.git>
7. git branch <branch name>
8. git branch
9. git checkout <branch name>
10. git diff <branch1> <branch2>
